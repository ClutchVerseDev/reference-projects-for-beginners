# ⏰ Python Alarm Clock

This is a simple Python program that lets you set an alarm using time input.

## 🚀 Features
- Set alarm in HH:MM:SS format
- Displays current time
- Alerts when time matches

## ▶️ How to Run
1. Run the Python file
2. Enter time in format HH:MM:SS
3. Wait for the alarm

## 📁 File
- alarm_clock.py
